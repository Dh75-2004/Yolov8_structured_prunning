# Yolov8_structured_prunning
In this project I took pretrained YOLOv8 classification model, model was trained for object detection task on hardware in real time all data collection labeling and data augumentation were done that data collection and training process is not parat of this project.Now let's see some statisstics of this yolov8 classification model it have acheived Top1 accuracy of 92.03% and no of parameters in this model are 14,51,098 trainable parameter and macs require for single pass of this model is 39,58,602.The objective of this project is do coarse grained
(Structured) prunning of yolo v8 classification model
